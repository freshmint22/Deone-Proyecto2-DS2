# Backend - DeOne Proyecto2 DS2

Instrucciones para correr el backend localmente.

1. Instalar dependencias

```powershell
cd backend
npm install
```

2. Copiar .env.example -> .env y completar variables

3. Ejecutar en dev

```powershell
npm run dev
```

4. Ejecutar tests

```powershell
npm test
```
